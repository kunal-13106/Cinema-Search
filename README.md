# 🎬 CineSearch

## 📌 Project Overview

CineSearch is a movie exploration web application that allows users to search and discover movies using real-time data from a public API. The app provides detailed information and interactive features to enhance user experience.

## 🎯 Purpose

This project aims to demonstrate:

* API integration using fetch
* Dynamic rendering of data
* Use of Array Higher-Order Functions
* Building responsive and user-friendly interfaces

## 🌐 API Used

* TMDB API (The Movie Database)

## 🚀 Features (Planned & Implemented)

* 🔍 Search movies by title
* 🎯 Filter movies by genre
* 📊 Sort movies by rating, popularity, or release date
* ❤️ Add movies to favorites
* 🎥 View movie details (poster, rating, overview)
* 🌙 Dark/Light mode

## 🛠️ Technologies Used

* HTML
* CSS / Tailwind CSS
* JavaScript (ES6+)

## ⚙️ Setup Instructions
