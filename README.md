# CineSearch
  > Discover Movies & TV Shows You'll Love

## About
CineSearch is a movie and TV show discovery app powered by The Movie
Database (TMDB) API. Search for films, explore trending titles, filter
by genre, sort by rating or release date, and save your watchlist.

## Features
- Search movies & TV shows by title (with debouncing)
- Filter by genre (Action, Comedy, Drama, Horror, etc.)
- Sort by popularity, rating, or release date
- View detailed movie modal with poster, overview, cast, trailer link
- Add to Watchlist / Favorites (saved in Local Storage)
- Browse Trending, Top Rated, Now Playing sections
- Dark mode / Light mode toggle
- Fully responsive across mobile, tablet, and desktop

## API Used
- TMDB API - https://www.themoviedb.org/documentation/api
  (Free API key — sign up at themoviedb.org)

## Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript (ES6+)
